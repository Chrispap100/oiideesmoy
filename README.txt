Οι Ιδέες Μου - Τελική Έκδοση

Περιλαμβάνει:
- Login με ρόλους (Super Admin, Admin, User)
- Καταγραφή ιδεών (χειροκίνητα ή με φωνή)
- Tags, κατηγορίες, καρφίτσωμα, AI έμπνευση
- Υπενθυμίσεις με ώρα και τοποθεσία
- Ιστορικό ενεργειών χρήστη
- Live Feed (δραστηριότητα σε πραγματικό χρόνο)
- Feedback χρηστών
- Εξαγωγή σε PDF/Excel
- Offline λειτουργία & Cloud Sync
- Αυτόματο Dark Mode
- Επαναφορά διαγραμμένων & κλωνοποίηση ιδεών
- Σχόλια & συνημμένα ανά ιδέα
- Ημερολόγιο προβολής ιδεών
- Μηνιαίο PDF report και email ειδοποιήσεις

Οδηγίες:
- Ανεβάστε τα αρχεία σε hosting πλατφόρμα (π.χ. Render, Netlify)
- Ο πρώτος χρήστης με login είναι ο Super Admin
- Επεκτείνεται εύκολα με Firebase, Supabase ή Node.js backend
